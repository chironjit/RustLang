# Brown Rust Tutorial Based on the Main Rust Tutorial

https://rust-book.cs.brown.edu

# Getting Set Up and Running

## Installation

`$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`

## Checking version

`$ rustc --version`

## Updating
`rustup update`

## Uninstalling
`$ rustup self uninstall`

# Quick Start
